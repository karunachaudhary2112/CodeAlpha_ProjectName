CodeAlpha Task-01: Web Scraping Internship 

Overview

This project performs web scraping on the CodeAlpha website using Python.
The script extracts headings, links, and table-like data from the webpage.

Technologies Used

- Python
- BeautifulSoup
- Requests
- Pandas
- Google Colab

Features

- Extracts headings (H1, H2, H3)
- Extracts all hyperlinks from the webpage
- Detects tables if available
- Saves webpage HTML for inspection

Website Scraped

https://www.codealpha.tech

Output

The script prints:

- List of headings
- Total number of links
- Table data (if available)

It also saves the webpage HTML as:
codealpha_homepage.html
