CodeAlpha Data Analytics Internship

Overview

This repository contains the projects completed as part of the CodeAlpha Data Analytics Internship.
The objective of these tasks is to demonstrate practical skills in web scraping, exploratory data analysis (EDA), and data visualization using Python.

---

Tools & Technologies

The following technologies and libraries were used in these projects:

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- BeautifulSoup
- Requests
- Google Colab

---

Task 1: Web Scraping

This task focuses on extracting data from the CodeAlpha website using Python-based web scraping techniques.

Key Features

- Extracts webpage headings (H1, H2, H3)
- Collects hyperlinks from the webpage
- Detects and extracts tables when available
- Saves the webpage HTML for further analysis

---

Task 2: Exploratory Data Analysis (EDA)

In this task, a Titanic-like dataset was generated and analyzed to understand patterns, distributions, and data characteristics.

Analysis Performed

- Dataset creation and loading
- Data inspection using "head()" and "describe()"
- Identification of missing values
- Basic statistical analysis
- Visualization of age distribution
- Survival analysis based on passenger class
- Detection of anomalies in fare values

---

Task 3: Data Visualization

This task focuses on creating meaningful visual representations of the dataset to uncover insights.

Visualizations Included

- Age distribution histogram with KDE
- Survival rate by gender
- Survival rate by passenger class
- Fare vs Age scatter plot with regression
- Dashboard-style combined visualization

---

Output Files

The project generates the following outputs:

- "demo_titanic_like.csv" – Generated dataset used for analysis
- Visualization plots saved as ".png" images
